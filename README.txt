If you've obtained this via the APress website, the best way to be up-to-date
with the book sources is via the GitHub site for the book:

	http://www.github.com/progrx

To check out the latest version of the sources, install MSysGit from
http://code.google.com/p/msysgit/, then run the following commands:

	git checkout git://github.com/ProgRx/EntireBook.git
	cd EntireBook
	git submodule init
	git submodule update
